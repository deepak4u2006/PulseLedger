# PulseLedger screenshots

Replace the placeholder PNGs in this folder with real Simulator captures (⌘S or **File → Save Screen**).

## Required captures

| File | Screen |
|------|--------|
| `dashboard.png` | Home dashboard — cards, balance, chart, transaction stack |
| `auth.png` | Auth flow — welcome, email, PIN, or biometrics step |
| `transaction-history.png` | Expanded transaction stack / recent list |
| `detail.png` | Transaction detail hero transition |

## Tips

1. Run on **iPhone 16** or **iPhone 16e** simulator (iOS 17+).
2. Use a consistent dark appearance (app uses `FintechTheme`).
3. **Screen recordings:** Simulator → **Record Screen** (or ⌘R in QuickTime) for README/GIF demos.
4. Recommended size: ~390×844 pt (@3x export optional); keep under 500 KB each for GitHub.

After adding images, the table in the root [README.md](../../README.md) will render automatically.
