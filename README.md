# PulseLedger

Personal finance dashboard — Clean Architecture, SwiftData offline cache, MVVM.


## Architecture
See diagram in repo root docs. Built with **SwiftUI**, **Swift Concurrency**, Clean-inspired layers.

## Run
1. Open `PulseLedger.xcodeproj` in Xcode 15+
2. Select an iPhone simulator → Run (⌘R)
3. Tests: ⌘U
